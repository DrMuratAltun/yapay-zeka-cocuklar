ML for Kids Nedir?

ML for Kids (Machine Learning for Kids), çocuklara makine öğrenmesi kavramlarını öğreten ücretsiz bir web platformudur. Metin, görüntü veya sayı verilerini kullanarak kendi modelinizi eğitebilir ve Scratch'te kullanabilirsiniz.

📋 Adım Adım Kılavuz
1

Hesap Oluştur / Giriş Yap

  • machinelearningforkids.co.uk adresine git
  • "Get Started" butonuna tıkla
  • "Try it now" ile hesapsız dene veya öğretmeninden sınıf hesabı al
💡 İpucu: Öğretmenin sınıf hesabı oluşturursa projelerini kaydedebilirsin!
2

Yeni Proje Oluştur

  • "+ Add a new project" butonuna tıkla
  • Projeye bir isim ver (ör: "Duygu Tanıma")
  • Veri türünü seç: Metin, Görüntü veya Sayı
  • "Create" butonuna tıkla
3

Eğitim Verisi Ekle (Train)

  • Projenin "Train" sekmesine git
  • "+ Add new label" ile sınıflar (etiketler) oluştur
  • Her sınıfa en az 5 örnek ekle
  • Metin projesi: Cümle yaz | Görüntü projesi: Resim yükle
⚠️ Örnekler çeşitli olmalı — hep aynı cümleyi yazma!
4

Modeli Eğit (Learn & Test)

  • "Learn & Test" sekmesine git
  • "Train new machine learning model" butonuna tıkla
  • Eğitimin tamamlanmasını bekle (birkaç dakika sürebilir)
  • Test alanına yeni örnekler girerek modeli dene
💡 Model iyi çalışmıyorsa daha fazla örnek ekleyip tekrar eğit!
5

Scratch'te Kullan (Make)

  • "Make" sekmesine git
  • "Scratch 3" butonuna tıkla
  • Scratch editörü açılacak — projeye özel YZ blokları hazır!
  • "recognise ... (label)" bloğunu kullanarak sınıflandırma yap
  • Sonuca göre karakter hareketleri, sesler veya görüntüler ekle
  • Yeşil bayrak ile projeyi çalıştır!

📝 Proje Notlarım

Projende ne sınıflandırdın? Kaç sınıf oluşturdun? Model ne kadar doğru çalıştı?
